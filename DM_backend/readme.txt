Чтобы запустить сервер нужно вызвать комманду java -jar server_dm-1.2.jar <путь до папки с генераторами>
пример:
java -jar server_dm-1.2.jar /home/cyansmoke/DM_backend/src/main/resources/generators/
слэш(/) в конце важен
